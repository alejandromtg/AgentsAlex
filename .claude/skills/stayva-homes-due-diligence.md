# Skill: Stayva Homes — Due Diligence Inmobiliaria Ribadeo

## Identidad del proyecto

**Stayva Homes** es un proyecto de inversión inmobiliaria en Ribadeo (Lugo, Galicia) liderado por el cliente. El objetivo es analizar propiedades singulares para determinar su viabilidad como:
- Vivienda de lujo singular
- Apartamentos turísticos (VUT)
- Hotel boutique
- Hotel de lujo / Relais & Châteaux

## Metodología de análisis por propiedad

Para cada propiedad analizar en este orden:

### 1. Ficha técnica (datos del anuncio)
- Precio pedido, m² construidos, m² parcela
- Nº plantas, habitaciones, baños
- Año construcción, estado (reformar/buen estado)
- Dirección exacta, referencia catastral

### 2. Análisis visual (desde fotos/capturas)
- Tipología arquitectónica (pazo, casona, chalet moderno, piso...)
- Estado real de conservación: fachada, interior, cubierta, instalaciones
- Calidad de vistas (ría, mar, montaña, casco, sin vistas)
- Jardín/terreno: tamaño estimado, uso actual, potencial
- Edificaciones auxiliares (casita, garaje, bodega, hórreo...)
- Señales de alerta: humedades, estructura comprometida, abandono

### 3. Análisis urbanístico (Rubén)
- Clasificación PGOM: casco histórico, ensanche, rústico, costas
- Verificar afección Ley de Costas (100m DPMT) si es zona litoral
- Verificar Red Natura 2000 si es zona estuario del Eo
- Edificabilidad restante en parcela: ¿se puede ampliar o construir anexo?
- Estado licencias: primera ocupación, VUT, obras
- Protección patrimonial: ¿está catalogada?

### 4. Análisis legal (Lola)
- Solicitar nota simple registral: cargas, hipotecas, embargos
- Verificar cédula de habitabilidad vigente
- Verificar certificado energético
- IBI al corriente (últimos 4 años)
- Cuotas comunidad al corriente (si aplica)
- Primera ocupación / licencia de actividad (si uso turístico u hotelero)

### 5. Modelo financiero — 4 escenarios (Javier)

#### Coste de reforma estimado
| Calidad | €/m² | Notas |
|---------|------|-------|
| Básica habitable | 900–1.200 | Reformar sin lujos |
| Media-alta residencial | 1.400–1.800 | Buenas calidades |
| Lujo / hotel boutique | 2.000–2.800 | Materiales nobles, diseño |
| Restauración patrimonial | 2.500–3.500 | Casco histórico protegido |

> Añadir 20–35% si hay protección patrimonial (uso obligatorio de granito, pizarra, madera tradicional).

#### Escenario A — Vivienda de lujo
```
Valor reformado = Precio compra + Coste reforma + Margen mercado
Comprador objetivo: familia alto patrimonio, extranjero, segunda residencia
Liquidez: baja. Tiempo de venta: 12–36 meses
```

#### Escenario B — VUT Premium (Airbnb/Booking)
```
Ingreso bruto anual = Nº suites × Tarifa media × Ocupación media × 365
Gastos operativos = 35–45% ingresos brutos
Rentabilidad neta = (Ingresos - Gastos) / Inversión total × 100
Referencia Ribadeo: tarifa alta temporada 150–300€/noche suite
Ocupación media anual realista: 55–65%
```

#### Escenario C — Hotel Boutique (8–14 hab.)
```
ADR (precio medio habitación): 120–220€
RevPAR = ADR × Ocupación media (55–70%)
Ingresos anuales = RevPAR × Nº hab. × 365
EBITDA margen = 25–35%
Rentabilidad sobre inversión total = EBITDA / Inversión
```

#### Escenario D — Hotel de Lujo / Relais & Châteaux
```
ADR objetivo: 300–600€
Ingresos complementarios: restaurante gastronómico, spa, eventos
EBITDA margen = 28–38%
Inversión total mínima viable: 3.500.000–5.000.000€
Activo con alta revalorización patrimonial
```

### 6. Precio objetivo de negociación (Carmen)
```
Valoración base = Valor suelo + Valor edificio en estado actual
Ajustes:
  + Prima vistas excepcionales: +15–30%
  + Prima singularidad arquitectónica: +10–20%
  - Descuento reforma integral: –25–40%
  - Descuento protección patrimonial (restricciones): –10–15%

Precio objetivo negociación = Valoración ajustada
Margen de negociación típico Ribadeo: 8–15% sobre precio pedido
```
